Usage: privacy-pools withdraw quote [options] <amountOrAsset> [asset]

Request relayer quote and limits without generating a proof

Arguments:
  amountOrAsset       Amount to withdraw (canonical first positional)
  asset               Asset symbol (canonical second positional,
                      case-insensitive)

Options:
  -t, --to <address>  Recipient address (recommended for an accurate fee quote)
  -h, --help          display help for command
Canonical order: privacy-pools withdraw quote <amount> <asset>

Tip: add --help-full for examples, safety notes, and JSON fields.
